To use (NODEMON installed globally required):

1. Clone the repo.
2. npm install
3. Fill strings in 'env_vars_tmp.js' file with your credentials
4. Change the name of 'env_vars_tmp.js' to 'env_vars.js'
4. npm test
5. npm start
6. Endpoint example: localhost:8000/api/city/all
